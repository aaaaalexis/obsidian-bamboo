# Bamboo

Optimize Obsidian editor word boundaries for CJK text (Han, Hiragana, Katakana, Hangul).

最佳化 Obsidian 編輯器針對 CJK 文字（漢字、平假名、片假名、諺文）斷詞邊界判定。

## Overview

- CJK-aware word selection / 支援 CJK 智慧斷詞
- Preserves Obsidian default behavior for non-CJK text / 非 CJK 文字保留原生行為
- Zero configuration (works immediately when enabled) / 零設定（啟用後立即生效）

---

This plugin was built entirely with **Claude Sonnet 4.6** and **GPT-5.3-Codex**.

此外掛程式完全由 **Claude Sonnet 4.6** 與 **GPT-5.3-Codex** 建構。
